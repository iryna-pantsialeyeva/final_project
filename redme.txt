В качестве финального проекта по первой части у вас будет:
Библиотека (добавление, удаление, редактирование записей о книгах, авторах, жанрах, отзывы, обзоры)

Требования к проекту:

Добавление и редактирование книг происходит при помощи внесения данных в консоли приложения
Для добавления - вносите название книги, жанр, имя автора. В приложении вы должны:
проверить нет ли такой книги в базе. Если книга есть - вернуть информацию о ней вместе с ID.
проверить нет ли жанра с таким именем. Если жанр есть - установить новой книге уже существующий жанр и вернуть информацию о книге
проверить нет ли автора с таким именем. Если автор есть - установить новой книге уже существующего автора и вернуть информацию о книге

Для редактирования - указываете ID книги, которую хотите изменить, новое название книги, новое имя автора, новый жанр. Логика по обновлению та же, что и при добавлении

Для удаления книги - отправляете только её ID;

Для получения одной книги - отправляете только её ID;

Работа с базой данных из приложения должна осуществляться по следующему принципу:
запросы в базу данных должны осуществляться с помощью методов классов Repository или Dao ( BookRepository, AuthorRepository, GenreRepository и т.д)
Методы классов репозиториев должны вызываться из соответствующих классов сервисов (BookService, AuthorService и т.д)
Репозитории и сервисы должны реализовывать соответствующих интерфейсов (BookRepositoryImpl implements BookRepository, BookServiceImpl implements BookService и т.д)
Для получения соединения создать класс DataSourceUtil в котором будет метод getConnection(), возвращающий объект соединения с БД и метод returnConnection(Connection conn), который будет возвращать соединение обратно в пул.
Все классы должны быть корректно разложены по соответствующим пакетам
